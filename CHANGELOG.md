# Changelog

All notable changes to ShopSphere will be documented in this file.

## [1.0.0] - 2026-01-08

### Added
- Initial release of ShopSphere
- Product list with responsive grid layout
- Product detail page with full information
- Shopping cart with add/remove/update functionality
- Mock checkout with form validation
- Category filtering (6 categories)
- Price range filtering ($0-$300)
- Price sorting (ascending/descending)
- Redux Toolkit for state management
- React Router for client-side routing
- Comprehensive test suite (10+ tests)
- Responsive CSS with mobile-first design
- Complete documentation (7 guides)
- Production build optimization (89 KB gzipped)

### Features
- Browse 8 mock products
- Filter products by category
- Filter products by price range
- Sort products by price
- Add items to cart
- Remove items from cart

- Form validation
